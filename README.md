# Simple Calculator

A simple command-line calculator written in Python that can perform basic arithmetic operations: addition, subtraction, multiplication, and division. It handles invalid inputs and prevents division by zero.  

## Features
- Addition
- Subtraction
- Multiplication
- Division (with zero-division error handling)
- User-friendly menu interface
- Continuous operation until the user chooses to exit

## Prerequisites
- Python 3.x installed on your system

## How to Use
1. Clone or download the repository.
2. Open a terminal or command prompt.
3. Run the script:

```bash
python calculator.py
